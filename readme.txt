README - คู่มือการใช้งานโปรแกรมรายการจาก Prompt
โปรแกรมนี้เป็นหน้าเว็บที่อนุญาตให้ผู้ใช้สามารถเพิ่มรายการต่าง ๆ ลงในรายการได้ โดยการกดปุ่ม "เพิ่มรายการ" แล้วกรอกข้อมูลที่ต้องการเพิ่มลงไป
วิธีใช้งาน:
    1 เปิดไฟล์ index.html ในเบราว์เซอร์ของคุณ (แนะนำให้ใช้ Google Chrome, Firefox หรือ Microsoft Edge)
    2 เมื่อหน้าเว็บโหลดขึ้นมา จะเห็นปุ่ม "เพิ่มรายการ" และส่วนแสดงรายการที่เพิ่ม
    3 เมื่อคุณคลิกปุ่ม "เพิ่มรายการ" จะมีหน้าต่าง prompt ให้กรอกข้อมูลรายการที่ต้องการเพิ่ม
    4 กรอกข้อมูลที่ต้องการ แล้วกด "ตกลง" (OK)
    5 รายการที่กรอกจะถูกเพิ่มลงในรายการด้านล่างปุ่ม
โครงสร้างไฟล์:
    • index.html : ไฟล์ HTML หลักที่เป็นหน้าเว็บแสดงผล
    • styles.css : ไฟล์ CSS ที่ใช้ในการจัดการรูปแบบการแสดงผล
    • script.js : ไฟล์ JavaScript ที่ใช้จัดการการเพิ่มและแสดงผลรายการ
ฟังก์ชันหลักในโปรแกรม:
    1 addItem() : ฟังก์ชันที่ใช้ในการเรียก prompt เพื่อให้ผู้ใช้กรอกข้อมูล และเพิ่มรายการลงในรายการ
    2 displayList() : ฟังก์ชันที่ใช้ในการแสดงรายการที่ถูกเพิ่มไว้ในหน้าเว็บ
ข้อจำกัด:
    • ผู้ใช้ไม่สามารถเพิ่มรายการว่าง (ข้อความว่าง) ลงในรายการได้
    • รายการจะถูกรีเซ็ตทุกครั้งที่รีเฟรชหน้าเว็บ เนื่องจากไม่มีการเก็บข้อมูลแบบถาวร
